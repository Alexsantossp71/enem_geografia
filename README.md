# Geografia Project

Este repositório contém a estrutura inicial para o projeto "Geografia". Ainda não há código implementado; este é apenas um esqueleto de ponto de partida.

## Estrutura

- `src/` – Diretório onde o código-fonte será colocado.
- `.gitignore` – Arquivo de configuração para ignorar arquivos desnecessários ao versionamento.

## Como começar

1. Clone o repositório.
2. Instale as dependências necessárias (quando houver).
3. Abra o arquivo `index.html` no navegador ou sirva o diretório usando um servidor local, por exemplo:
   ```bash
   # Python 3
   python -m http.server 8000
   # Ou com npm http-server (se instalado)
   npx http-server -p 8000
   ```
   Acesse http://localhost:8000 no seu navegador.
4. Comece a desenvolver dentro da pasta `src/`.
