# Geografia para o ENEM

Site educativo para estudo de geografia focado no Exame Nacional do Ensino Médio (ENEM), baseado no Capítulo 2 do caderno de Ciências Humanas.

## 🎯 Objetivo

Fornecer uma plataforma interativa para estudo de geografia com foco nos conteúdos mais cobrados no ENEM, utilizando jogos educativos e materiais estruturados.

## 🏗️ Estrutura do Site

### Páginas Principais
1. **index.html** - Página inicial com visão geral dos tópicos
2. **aula_capitulo2.html** - Aula completa sobre o Capítulo 2: Ciências Humanas
3. **Indice_aulas.html** - Índice organizado por temas do ENEM

### Jogos Educativos
- **Flashcards** (`games/flashcards/`) - Cartões de memorização
- **Jogo da Memória** (`games/memory/`) - Pareamento de conceitos
- **Verdadeiro ou Falso** (`games/truefalse/`) - Quiz rápido
- **Mapa do Brasil** (`games/maps/brasil.html`) - Interativo com regiões
- **Mapa Mundial** (`games/maps/mundo.html`) - Geopolítica global
- **Caça-palavras** (`games/crossword/`) - Vocabulário geográfico

### Recursos
- **PDF do Capítulo 2** - `caderno_ciencias_-humanas.pdf`
- **Backups** - Arquivos compactados com versões anteriores

## 🚀 Como Executar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/geografia.git
   ```

2. Abra o arquivo `index.html` em qualquer navegador moderno

3. Ou execute um servidor local:
   ```bash
   python -m http.server 8000
   ```
   Acesse: http://localhost:8000

## 🌐 GitHub Pages

O site está configurado para ser hospedado no GitHub Pages:

1. Acesse as configurações do repositório no GitHub
2. Vá para "Pages" na sidebar
3. Configure a fonte como "Deploy from a branch"
4. Selecione o branch `main` e pasta `/ (root)`
5. Salve e aguarde a implantação

URL do site: `https://seu-usuario.github.io/geografia/`

## 🎨 Design

## 📚 Conteúdo Educativo

O site cobre os principais tópicos de geografia para o ENEM:

### Geografia Física
- Relevo e formações geológicas
- Clima e biomas
- Hidrografia e recursos hídricos

### Geografia Humana
- População e demografia
- Urbanização e metrópoles
- Economia e desenvolvimento

### Geopolítica
- Relações internacionais
- Conflitos e organizações mundiais

### Geografia do Brasil
- Regiões brasileiras
- Economia brasileira
- Desafios ambientais

### Meio Ambiente
- Sustentabilidade
- Mudanças climáticas

### Cartografia
- Interpretação de mapas
- Sistemas de coordenadas
